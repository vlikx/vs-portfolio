================================================
  HOW TO ADD PROJECTS
================================================

1. DROP YOUR IMAGE HERE
   - Supported: .jpg, .jpeg, .png, .gif, .webp, .avif
   - The filename becomes the title:
     "neon-genesis.jpg" → "NEON GENESIS"
     "product_render.png" → "PRODUCT RENDER"

2. ADD A DESCRIPTION (OPTIONAL)
   Create a .txt file with the same name:
   
   my-project.jpg
   my-project.txt  ← description file
   
   First line = Category (e.g., "3D Visualization")
   Rest = Description text

3. EXAMPLE
   ─────────────────────────────
   File: cool-project.txt
   ─────────────────────────────
   3D Visualization
   A futuristic cityscape rendered in Blender.
   Created for a client in the gaming industry.
   ─────────────────────────────

That's it! Projects auto-appear in the grid.
